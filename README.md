# movies-catalog

Первый шаг - клонирование проекта и установка зависимостей:

```
git clone git@github.com:serega-lesnik/movies-catalog.git
npm install
```

Запуск проекта в *dev* окружении:

```
npm run dev
```

Сборка (build) *production*:

```
npm run prod
```
